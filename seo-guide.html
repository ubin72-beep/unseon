구글 서치콘솔 등록 Google Search Console
구글은 전 세계 최대 검색엔진입니다. 서치콘솔에 등록하면 구글 검색 결과에 빠르게 노출되고, 클릭수·노출수·순위를 실시간 모니터링 할 수 있습니다.
1

Google Search Console 접속 & 로그인

구글 계정(sajuon@gmail.com 권장)으로 로그인 후 속성 추가를 클릭합니다.

Google Search Console 바로가기
2

속성 추가 → URL 접두어 선택

속성 유형에서 URL 접두어를 선택하고 아래 URL을 입력합니다.

https://unseon.co.kr/
3

HTML 태그 인증 방법 선택

소유권 확인 방법에서 "HTML 태그"를 선택합니다. 아래와 같은 코드가 표시됩니다.

<meta name="google-site-verification" content="여기에_인증코드" />

이 인증코드(content 값)를 아래에 입력하면 index.html에 자동 반영됩니다.

현재 설정값: naver_unseon_verification_code

4

확인 버튼 클릭 → 소유권 인증 완료

서치콘솔에서 "확인" 버튼을 누르면 자동으로 인증이 완료됩니다. (배포 후 1~2분 내 인식)

5

사이트맵 제출

서치콘솔 왼쪽 메뉴 → Sitemaps → 아래 URL 입력 후 제출

https://unseon.co.kr/sitemap.xml

제출 후 구글이 사이트를 크롤링하는 데 1~7일 소요됩니다.

6

색인 생성 요청 (빠른 노출)

서치콘솔 상단 검색창에 아래 URL을 입력 후 "색인 생성 요청"을 클릭합니다.

https://unseon.co.kr/ https://unseon.co.kr/chat.html

이 작업으로 구글 크롤링 속도가 크게 빨라집니다.

1

카카오페이 사업자 가맹 신청

  1. 카카오 비즈니스 접속 → 로그인
  2. 서비스 신청 → 카카오페이 결제 선택
  3. 사업자 정보 입력 (사업자등록번호: 537-08-03349)
  4. 통신판매업 신고증 업로드 (준비 후 진행)
  5. 심사 승인 후 CID (가맹점 ID) 발급받기
심사 기간: 영업일 기준 3~7일 / 문의: kakaopay@kakao.com
2

js/kakaopay.js 설정값 변경

파일 상단의 두 줄만 수정하면 됩니다:

// 수정 전 (테스트)
const IS_TEST_MODE = true;
const KAKAO_CID = IS_TEST_MODE ? 'TC0ONETIME' : 'C여기에_실제_CID_입력';

// 수정 후 (실서비스)
const IS_TEST_MODE = false;
const KAKAO_CID = IS_TEST_MODE ? 'TC0ONETIME' : '실제_CID_여기입력';
보안 주의: 실서비스에서는 카카오 Admin API Key가 프론트엔드에 노출되면 안 됩니다. 결제 준비(ready) API 호출은 반드시 서버사이드에서 처리해야 합니다. 현재 구조는 서버 없는 환경용 참고 코드입니다.
3

리다이렉트 URL 확인

카카오페이 결제 완료/실패/취소 후 이동할 URL이 정확히 설정되어 있는지 확인하세요:

구분 URL 상태
approval_url https://unseon.co.kr/payment-complete.html?result=success ✅ 설정됨
fail_url https://unseon.co.kr/payment-complete.html?result=fail ✅ 설정됨
cancel_url https://unseon.co.kr/payment-complete.html?result=cancel ✅ 설정됨
카카오페이 파트너 사이트에서도 동일한 URL을 Whitelist에 등록해야 합니다.
4

5

통신판매업 신고 (필수)

전자상거래법 위반 주의: 온라인 결제 서비스는 통신판매업 신고가 의무입니다.
  1. 공정거래위원회 사이버몰 또는 정부24 접속
  2. 통신판매업 신고 → 사업자 정보 입력
  3. 신고필증 발급 후 사이트에 신고번호 표시
  4. pricing.html footer에 신고번호 추가 필요

실서비스 전환 체크리스트

  • ☐ 사업자등록증 확보
  • ☐ 통신판매업 신고 완료 + 신고번호 획득
  • ☐ 카카오페이 가맹 신청 → 심사 통과 → CID 발급
  • ☐ js/kakaopay.js 에서 IS_TEST_MODE = false 변경
  • ☐ KAKAO_CID 에 실제 CID 입력
  • ☐ 카카오페이 파트너 사이트에서 허용 도메인 등록
  • ☐ 실결제 테스트 1회 진행 (소액 결제 → 환불 확인)
  • ☐ 구매 내역 / 환불 정책 페이지 최신화
