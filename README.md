> 궁금할 때 바로 묻는 사주 · 타로 · 궁합 · 이름짓기 AI 상담 서비스

## 🌐 주요 페이지

| URL | 역할 |
|-----|------|
| `index.html` | 메인 랜딩 (매출형 카피 + 카테고리 + 후기 + FAQ) |

## ✅ 완성된 기능

### 메인 랜딩 (index.html)
- 매출형 카피 히어로 섹션 (질문 직접 입력창 + 빠른 칩)
- 신뢰 수치 바 (상담 건수 · 만족도 · 분야 수 · 질문당 가격)
- **핵심 카드 TOP 4** (연애운 / 궁합 / 사업운·재물운 / 직업상담)
- 2026 병오년 시즌 배너
- 심화 상담 카드 6종 (이름짓기 · 개명 · 상호명 · 타로 · 개업 · 점성술)
- 전체 카테고리 탭 (30가지 분야)
- 충전 플랜 미리보기
- 후기 6건 (localStorage로 관리)
- FAQ (localStorage로 관리)
- CTA 하단 배너

### AI 채팅 상담 (chat.html)
- ChatGPT 스타일 대화형 UX
- **30가지 상담 카테고리** 사이드바
- 카테고리별 맞춤 AI 응답 (연애운·사업운·직업·이름짓기 특화 응답)
- 타이핑 인디케이터 (1.2~2.2초 딜레이)
- 포인트 실시간 차감 (100P / 200P / 300P)
- 포인트 부족 시 모달 팝업
- 이용 내역 자동 저장

### 포인트 충전 (pricing.html)
- 3단계 플랜 카드 (베이직 / 스탠다드 / 프리미엄)
- 충전 확인 → 완료 시뮬레이션 플로우
- 상담 유형별 차감 기준표
- 이용 내역 테이블

### 관리자 대시보드 (admin.html)
- **대시보드**: 통계 카드 (충전/차감/상담 건수/보유 포인트) + 최근 내역
- **배너·카피 관리**: 히어로 제목·뱃지·서브텍스트·배너 문구 수정
- **카테고리 관리**: 아이콘·이름·차감P·탭·질문예시 CRUD
- **요금 정책 관리**: 플랜별 금액·포인트·보너스·차감 기준 설정
- **후기 관리**: 별점·내용·작성자·분야 CRUD (최대 6개 표시)
- **FAQ 관리**: 질문·답변 CRUD
- **이용 내역 조회**: 전체 내역 테이블 + 초기화
- **포인트 조작**: 빠른 충전·직접 설정·초기화·데이터 내보내기(JSON)
